# bitchan

bittorrent imageboard 

## build deps

* go 1.13.x
* yarn
* GNU Make
* git

## building

usage:

    $ git clone https://github.com/majestrate/bitchan 
    $ cd bitchan
    $ make mistake
    $ go get -u github.com/majestrate/bitchan

running:

    $ bitchan your.domain.tld
    
uses port 8800 by default
